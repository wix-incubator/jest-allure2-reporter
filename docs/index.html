Introduction

Thanks for choosing Jest Allure 2 Reporter!

Whether you're a developer, QA professional, or someone involved in unit, integration, or end-to-end testing, this tool will make your testing process more efficient, organized, and insightful.

What is jest-allure2-reporter?​

jest-allure2-reporter is a reporting toolkit for Jest, a widely popular testing framework in the JavaScript ecosystem.

It consists of several packages:

  • jest-allure2-reporter - the reporter itself, which is responsible for collecting test results and generating a report;
  • jest-allure2-reporter/annotations - a set of functions and decorators to add additional metadata to your tests;
  • Environment packages, to enable the annotations and provide additional test data:
    • jest-allure2-reporter/environment-node - for Node.js tests;
    • jest-allure2-reporter/environment-jsdom - for browser tests;
    • jest-allure2-reporter/environment-extend - for advanced use cases where you need to extend your already customized test environment.

The produced reports look like directories with multiple JSON files. They can be viewed using the allure command-line tool, a flexible, feature-rich reporting tool that provides clear and concise test result representation.

Why one more Allure reporter for Jest?​

With Jest's evolution from Jasmine to its own Circus test runner in version 27.0.0, the testing landscape shifted dramatically. While this change brought new opportunities for enhanced testing, it also created a void: existing Allure reporters1, designed for Jasmine, fell short of leveraging Jest's new capabilities.

While community-made reporters sought to bridge this gap234, they were not without limitations. Issues with irregular maintenance, limited reach, and significant feature gaps were prevalent. Most critically, they took a non-idiomatic approach by acting as extensions to Jest's testEnvironment, which led to a critical shortcomings: e.g., inability to report early test issues like SyntaxError or test environment setup failures.

Enter Jest Allure 2 Reporter​

Jest Allure 2 Reporter is not just another Allure reporter; it's a powerful tool tailor-made for the modern Jest ecosystem. By effectively capturing data from Jest's test environment, it provides comprehensive insights into your tests.

Flexible by design, it works seamlessly regardless of you've extended your test environment or not.

Jest Allure 2 Reporter gives you the necessary insights for your testing tasks in an easy-to-understand format. Easy to understand, straightforward to adopt, and flexible to work with – it's the Jest Allure reporter you've been waiting for.

Take your testing game to the next level with Jest Allure 2 Reporter today!
