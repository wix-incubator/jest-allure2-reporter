# Defect categories

Classify your test failures into different categories such as Product defects, Test defects and any custom categories fit for your project, e.g.:

```js title="jest.config.js"
/**
 * @type {import('@jest/types').Config.InitialOptions}
 */
module.exports = {
  // ... your jest config
  reporters: [
    'default',
    ['jest-allure2-reporter',
      /** @type {import('jest-allure2-reporter').JestAllure2ReporterOptions} */
      {
        categories: [
          {
            name: "Visual Regression",
            matchedStatuses: ["failed"],
            messageRegex: "Expected .*to match a screenshot.*"
          }
        ]
      }
    ],
  ],
};
```

This feature gives you the flexibility to organize the discovered defects in a way that makes the most sense for your project.
